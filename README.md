# R. U. Hacking? Website


[![GitHub license](https://img.shields.io/github/license/RUHacking/RUHWebsite.svg)](https://github.com/RUHacking/RUHWebsite/blob/develop/LICENSE)


### Installation
```
yarn
```

### Development
Starts a development server on localhost:8000
```
nvm use
yarn dev
```

### Lint and format
```
yarn format
yarn lint
```

### Build Prod Copy
```
yarn build
```
